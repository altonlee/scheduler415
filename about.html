Midterm Brainstorming

These are some ideas I brainstormed for this upcoming project. I will be working alone. I chose the school schedule maker, now dubbed Scheduler 415.

school schedule maker

  • users can range from students to professionals, to regular people that need a simple but pretty schedule form
  • freecollegeschedulemaker is a really good app, but i think the styling of the app is a bit outdated
  • will use HTML and JS React, Semantic UI for styling

data visualizer

  • input data into a form, and outputs a graph that visualizes that data
  • excel can create graphs based on data
  • will use HTML and JS React, Semantic UI for styling

idea generator

  • input two subjects, and will randomly output a string, which will hopefully inspire an idea from you
  • there are generators online, like pungenerator
  • will probably use HTML and JSQuery
